# Demonstration Assets: Advisor UI & CLI Scans

This folder contains the visual and data demonstrations validating the Flang Modernization Advisor system's functionality.

---

## 1. Interactive Web Dashboard Screenshots

### Dashboard Overview
The main interface displaying the Emerald & Obsidian theme, the file health score, the modernization checklist findings list, the code viewer, and the refactoring preview.
![Overall View](Overall%20View%20.png)

### Alerts & Code Line Highlighting
Clicking on any modernization card in the checklist highlights the specific line range in the code panel. In multi-file analysis, it automatically switches the file dropdown to the matching source file and centers the view on the target line.
![Alerts and Highlighting](Alerts%20and%20Highlighting.png)

### Modernization Recommendations
Detailed side-by-side comparison panels showing the legacy Fortran construct (left) and the proposed modern Fortran replacement (right).
![Recommendation Results](Recommendation%20Results.png)

## 2. Command Line Interface (CLI) Scan
