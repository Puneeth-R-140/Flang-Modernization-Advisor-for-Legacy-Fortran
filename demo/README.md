# Demonstration Assets: Advisor UI and CLI Scans

This folder contains the visual and data demonstrations validating the Flang Modernization Advisor system's functionality.

## 1. Interactive Web Dashboard Screenshots

The main interface displaying the Emerald and Obsidian theme, the file health score, the modernization checklist findings list, the code viewer, and the refactoring preview.

Clicking on any modernization card in the checklist highlights the specific line range in the code panel. In multi-file analysis, it automatically switches the file dropdown to the matching source file and centers the view on the target line.

Detailed side-by-side comparison panels showing the legacy Fortran construct (left) and the proposed modern Fortran replacement (right).

## 4. How to Reproduce the Demos

### On Linux / Inside WSL Terminal (AST parse-tree visitor active)

1. **Build the static analyzer**:
   ```bash
   ./build.sh
   ```
   Detects LLVM Flang 22 and compiles with `-DUSE_FLANG_PARSER`, outputting `build_wsl/flang-modernization-advisor`.

2. **Generate the JSON analysis report**:
   ```bash
   ./run.sh tests/enterprise_simulation.f90 --output demo/Demo_report_new.json
   ```

3. **Start the interactive web dashboard**:
   ```bash
   python3 app.py
   ```
   Open your browser at http://localhost:8000.

### From Windows Host Terminal using WSL

1. **Build the static analyzer**:
   ```powershell
   wsl ./build.sh
   ```

2. **Generate the JSON analysis report**:
   ```powershell
   wsl ./run.sh tests/enterprise_simulation.f90 --output demo/Demo_report_new.json
   ```

3. **Start the interactive web dashboard**:
   ```powershell
   python app.py
   ```
   Open your browser at http://localhost:8000.

### On Windows Native Terminal (text-based preprocessor only)

1. **Build the static analyzer**:
   ```powershell
   ./build.sh
   ```
   Outputs `build/Release/flang-modernization-advisor.exe`.

2. **Generate the JSON analysis report**:
   ```powershell
   ./run.sh tests/enterprise_simulation.f90 --output demo/Demo_report_new.json
   ```

3. **Start the interactive web dashboard**:
   ```powershell
   python app.py
   ```
   Open your browser at http://localhost:8000.
